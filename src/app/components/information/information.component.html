Acerca del proyecto


Motivación

Este proyecto es desarollado como parte del proceso de sustentación de tesis para optar por el titulo de ingeniería de Sistemas en la Universidad Francisco de Paula Santander

Principios

Este proyecto aplica un estilo de arquitectura orientado a los microservicios, conectado a un servicio web desarrolado previamente

Tecnologías

En el proyecto se emplean frameworks de desarrollo como es el caso de Angular, Flask y ... para la creación de los microservicios, los cuales son desplegados en la nube de AWS
